# Terraform Landing Zone for OVHcloud

This repository contains Terraform modules for deploying and managing a Landing Zone in OVHcloud.

It provides reusable infrastructure modules for common workloads and platform components, including:

- Networking
- Storage
- Virtual Machines
- Kubernetes
- Identity and access configuration
- Shared platform services

The repository also includes CI/CD workflows for automated infrastructure deployment and environment management.

A structured hierarchy is used to separate customer environments such as:

- Production
- Test
- Development

The goal of this repository is to serve as a reusable template and foundation for onboarding and deploying new customers on OVHcloud.
